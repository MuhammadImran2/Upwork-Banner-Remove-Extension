Upwork Banner Remove Extension

How to use this extension:

  1. Create a new folder for the extension files
  2. Save the manifest.json and content.js files in that folder
  3. Open Chrome and go to chrome://extensions/
  4. Enable "Developer mode" (toggle in top right)
  5. Click "Load unpacked" and select the extension folder
  6. The extension will automatically remove the announcement div from Upwork
